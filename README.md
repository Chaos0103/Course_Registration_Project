# 개인 프로젝트
* 수강신청시스템

## 개발환경
* Java
* Apache Tomcat 9.0
* HTML, CSS, JavaScript
* Spring Boot 2.6.6
* QueryDSL 5.0.0
* DB 서버: H2

## 핵심기능
통합 페이지
1. 학생 가입
2. 학생 목록
3. 교수 가입
4. 교수 목록

학생 페이지
1. 수강 신청/조회/취소
2. 관심과목 담기/조회/삭제
3. 당학기 성적 조회
4. 기이수 성적 조회
5. 강의시간표 조회

교수 페이지
1. 강의 등록/조회
2. 성적 등록
